# dld-apa

MINI-BC100介绍
dldapa

#### 软件架构
JAVA8.0
spring-Boot
mybatis-HikariCP
RXTX
Mysql8.0
websocket



#### 安装教程

1.  安装mysql8.0
2.  安装jdk/jre
3.  下载apa.jar，在文件夹打开cmd，用javaw -jar +jar文件路径打开
